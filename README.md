Un portafolio web de alto nivel, diseñado para demostrar que la excelencia técnica y estética en la web no dependen de herramientas complejas, sino del dominio profundo de los fundamentos.

---

## ✨ Características Principales

- **Arquitectura Glassmorphism**: Interfaz moderna con efectos de cristal puro y desenfoque dinámico (Pure Glass Mode).
- **Experiencia Inmersiva**: Efectos de paralaje profundo, cursor personalizado con resplandor reactivo y fondo de red neuronal interactivo.
- **Accesibilidad Avanzada (A11y)**:
  - **Asistencia por Voz**: Sistema de lectura dinámica integrado para una navegación inclusiva.
  - **Modos de Color**: Filtros específicos para daltonismo (Protanopia, Deuteranopia, Tritanopia).
  - **Contraste y Zoom**: Herramientas nativas para ajustar la legibilidad del contenido en tiempo real.
- **Localización (i18n)**: Soporte completo para Español e Inglés con persistencia de preferencias.
- **Animaciones Premium**: Transiciones fluidas de secciones, efectos 3D de inclinación (Tilt) en tarjetas y scroll suave orquestado con **GSAP** y **Lenis**.
- **Detalles de Autor**: Avatar interactivo con múltiples estados, marquee dinámico en el título del navegador y sistema de scrollytelling.

## 🛠️ Stack Tecnológico

Este proyecto ha sido construido utilizando tecnologías base para maximizar el rendimiento, la velocidad de carga y el control total sobre la experiencia del usuario:

- **HTML5**: Estructura semántica avanzada y optimizada para SEO.
- **Tailwind CSS**: Estilizado moderno mediante configuración personalizada y utilidades de diseño atómico.
- **JavaScript (Vanilla / jQuery)**: Lógica de interacción robusta, gestión de estados y orquestación de APIs del navegador.
- **Core Libraries**:
  - **GSAP (GreenSock)**: El estándar de oro para animaciones web profesionales.
  - **Lenis**: Para un desplazamiento (smooth scroll) natural y fluido.
  - **Leaflet**: Mapas interactivos personalizados.

## 🚀 Despliegue y Desarrollo

Al no depender de procesos de compilación pesados, la iteración es inmediata:

1. **Clonar**:
   ```bash
   git clone https://github.com/wydex28/dev-portfolio.git
   ```
2. **Ejecutar**:
   Abre el archivo `index.html` directamente en tu navegador o utiliza una extensión de servidor local ligero (como *Live Server* en VSCode).

## 🎨 Inspiración y Créditos

Este proyecto es una evolución personal y técnica que rinde homenaje a excelentes trabajos de la comunidad:

- **vCard Template**: El diseño base está inspirado en el popular [vCard Personal Portfolio](https://github.com/codewithsadee/vcard-personal-portfolio), adaptado completamente a un estilo moderno, modular y reactivo.
- **Dracula Theme**: La estética visual y paleta de colores se basan en el ecosistema [Dracula Theme](https://draculatheme.com/), con una versión personalizada para alto contraste neón.

## 💡 El Manifiesto "Back to Basics"

Este proyecto es un testimonio de una filosofía fundamental: **No se necesitan frameworks pesados ni build-chains excesivamente complejas para entregar un trabajo de alto nivel, profesional y visualmente impactante.**

En un mundo saturado de entornos de desarrollo que abstraen la web, este portafolio vuelve a las bases (**Vanilla HTML, CSS y JS**) para demostrar que el dominio técnico y la atención al detalle son las verdaderas herramientas del éxito. Si puedes imaginarlo y conoces los fundamentos, puedes construirlo sin obstáculos.
